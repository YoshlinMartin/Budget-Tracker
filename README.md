# Budget-Tracker
A budget tracker app that allows the user to login and register using firebase authentication. The app allows you to create a category of expenses, give the expense a title, the amount, the date and a description of the expense. As well as allows you to upload a picture from your gallery. You can then save the entry. You can also view those expenses in the from of a pie chart or a bar graph. You can then filter those expenses by date and it will show you all the information that the user has entered.
